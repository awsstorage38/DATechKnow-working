Amazon Web Services

It's a comprehensive, evolving cloud computing platform provided by Amazon that includes a mixture of infrastructure as a service (IaaS), platform as a service (PaaS), and packaged software as a service (SaaS) offerings.

GitHub

GitHub is a web-based platform used for version control and collaboration in software development projects. It allows developers to host and review code, manage projects, and build software alongside millions of other developers.

HTML5

HTML5, or Hypertext Markup Language version 5, is the latest evolution of the standard that defines HTML. It's the fifth revision of the HTML standard and introduces new features, elements, and behaviors for web development.

Linux

Linux is an open-source operating system kernel originally created by Linus Torvalds. It forms the core of many different distributions, or versions, of the Linux operating system, which is widely used in servers, supercomputers, and embedded systems

Ansible

Ansible is a powerful open-source automation tool used for configuration management, application deployment, and task automation. With its simple, human-readable syntax, Ansible allows you to easily automate repetitive tasks across multiple servers, making system administration more efficient and scalable.

Storage

Storage refers to the process of storing and managing digital data on computer systems or other storage devices. It involves organizing data in a structured manner to facilitate easy access, retrieval, and manipulation

Teraform

Terraform is an open-source infrastructure as code software tool created by HashiCorp. It allows users to define and provision data center infrastructure using a high-level configuration language known as HashiCorp Configuration Language (HCL), or optionally JSON
